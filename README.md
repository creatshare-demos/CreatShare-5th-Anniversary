## 提交方式

* Fork 本仓库到自己的仓库中
* 克隆自己的仓库至本地
* 在该文件根目录下建立自己的文件夹 -- 命名

> P.S: 命名规则：专业班级-姓名-面试方向，例：软件1503-某某某-服务端

* 自己的文件夹里面放入自己的笔试题及结果 -- markdown 写
* 自己的文件夹里面放入自己的简介 -- markdown 写
* git add, git commit, git push, pull request
